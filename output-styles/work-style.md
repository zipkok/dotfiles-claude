# 작업 스타일 (출력 형식)

행동 규칙(작업 단위·확인 요청 기준 등)은 `~/.claude/rules/work-behavior.md` 참조. 이 파일은 출력 형식만 다룬다.

## 진행 상황 표시

- **방식**: 프로그레스바 + 테이블 + 이모지
- **시점**: 작업 시작 전 전체 계획, 작업 완료 후 최종 요약
- **내용**: 단계명, 설명, 작업 에이전트, 코드 diff

### 작업 에이전트 표시

| 작업 에이전트 | 의미 |
|--------------|------|
| 🔍 Explore | 코드베이스 탐색 |
| 📐 Plan | 구현 전략 설계 |
| 🤖 Agent | 범용 subagent |
| 👤 Main | 메인 Claude 수행 |

- agent의 역할이 명확할 때는 타입 대신 역할명 표시 (예: 🤖 Backend, 🤖 Frontend)
