# JD Analyze

`/Users/woobs/Repository/jd-analysis/jds/` 폴더의 JD 텍스트 파일들을 읽고 분석하여 결과를 저장합니다.

## 사용법

- `/jd-analyze` — 전체 JD 분석 후 결과 저장
- `/jd-analyze {회사명}` — 특정 회사 JD만 분석

## 동작 방식

다음 순서로 실행합니다:

1. **이력서 읽기** — `resume.txt` 파일로 현재 보유 기술/경험 파악
2. **JD 파일 수집** — `jds/` 폴더의 모든 `.txt` 파일 목록 확인
3. **JD 내용 읽기** — 각 파일의 내용을 읽어 직무, 요구사항, 우대사항 파악
4. **기술 스택 분석** — 공통으로 등장하는 기술/툴/언어 빈도 집계
5. **부합도 분석** — 이력서 기준으로 각 JD 요구사항 충족 여부 평가
6. **공부 방향 제안** — 갭 기반 우선순위 학습 로드맵 제안
7. **결과 저장** — `analysis/YYYY-MM-DD_analysis.md` 파일로 저장

